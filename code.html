Concrete Jungle

Description
Making your home into something extraordinary doesn't have to be difficult or expensive. With a passion for simple, yet beautiful design with only concrete the people at (the fictive) company Concrete Jungle wanted a place to reach out with their products to new and old costumers. To do that, a prototype of a responsive webshop was made where it is possible to see items, learn more about them and also see how it looks when the items are in the cart.

Specifications
The project was done in Javascript, HTML and CSS and the program and design was all made by me. The project was the final assignment in a course in responsive webb design at Kristianstad University.

Quiz Columbus

Description
Learning should be fun and one way to make it more fun is simply making a game out of it. So if you or your kid wants to learn a little bit more about the adventureor Christopher Columbus this is the game for you. With four options and 10 seconds to answer each question about this explorer you will get a fun, yet informative game where you can compete with yourself or others by comparing the scores in the end of the game.

Specifications
The project was done in Python and the program and design was all made by me. The project was a project in a course in multimedia programming at Stockholm University.

Petshop

Description
